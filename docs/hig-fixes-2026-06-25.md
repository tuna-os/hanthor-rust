# GNOME HIG Fixes Applied

## Fix 1: Toolbar Icons (suite-common/src/lib.rs)
- Problem: Toolbar buttons showed text labels even when symbolic icons were set
- Fix: Removed redundant `set_label()` calls for `-symbolic` icon names, added `image-button` CSS class
- Affects: All 3 apps (Letters, Decks, Tables)

## Fix 2: Page Rendering (letters/src/window.rs)
- Problem: In dark mode, textview background was black, creating black rectangle inside white page
- Fix: Added CSS `textview, textview text, scrolledwindow { background: transparent; }`
- Affects: Letters

## Fix 3: Sidebar Controls (decks/src/window.rs)
- Problem: Sidebar add/delete/move buttons were tiny and frameless
- Fix: Set `has_frame(true)` and `size_request(36, 36)`
- Affects: Decks
